# Simple Calculator (Task 1)

This is a basic C++ console application that performs arithmetic operations. It's part of the Tutorial 1 engineering assignment.


## 🚀 Features

- **Addition (+)**: Adds two integers.

- **Subtraction (-)**: Subtracts the second integer from the first.

- **Multiplication (*)**: Multiplies two integers.

- **Division (/)**: Divides two integers and provides a `double` precision result.

- **Remainder (mod)**: Calculates the modulo of two integers.


## 📝 Math Logic

- Input: Two integers ($x1, x2$)

- Floating-point division: `(double)x1 / x2` to ensure decimal accuracy.


## 🧪 Example Test Cases

- **Case 1**: (8, 5) 

  - $8 + 5 = 13$

  - $8 / 5 = 1.6$

- **Case 2**: (9, 8)

  - $9 \text{ mod } 8 = 1$


## 🖥️ How to Run

1. Compile: `g++ prog1.cpp -o prog1`

2. Execute: `./prog1`

3. Enter `-1` to safely exit the program.
